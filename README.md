# Dashboard_31dias_de_dados
Nesse desafio foi desenvolvido um dashboard de 31 dias de dados 
O foco era responder perguntas sobre um site:

📊Qual era a quantidade de visitas por dia?
📊Qual tráfego trazia mais visualizações?
📊Qual era a taxa de conversão por dia?
📊Quantidade total de visitas únicas?
📊Total de taxa de conversão
📊Receita total?
📊 Mediana de visitas?
